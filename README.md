# Site Kassi

Site institucional para a psicologa clinica Kassiane Lins, com foco em acolhimento, informacoes claras e contato rapido via WhatsApp e Instagram.

## Para quem este projeto serve
- Psicologa/gestora do consultorio: apresenta servicos e facilita o contato.
- Pacientes: encontram informacoes, abordagem e canais de contato.
- Marketing/design: atualiza textos, imagens e identidade visual.
- TI/host: publica o site sem backend ou build.

## O que esta pronto
- Pagina unica com as secoes: Sobre Mim, Minha Abordagem, Servicos e Contato.
- Header com logo, nome e informacoes profissionais.
- Slideshow com frases de impacto.
- Animacao de texto por linha e efeitos suaves.
- Botoes flutuantes para WhatsApp e Instagram.
- QR code do WhatsApp.
- Medicao de cliques para WhatsApp/Instagram (Google Ads tag).
- Layout responsivo para celular e desktop.

## Fluxo para o visitante
1. Entra no site e conhece a profissional.
2. Le a abordagem e os servicos.
3. Clica no WhatsApp ou Instagram para iniciar o contato.

## Como visualizar
- Abra `index.html` no navegador.
- Se preferir, use um servidor local (ex.: extensao Live Server).

## Como personalizar (sem programar muito)
- Textos e links: edite `index.html`.
- Imagens (logo, perfil, QR, banners): troque arquivos em `imagens/`.
- Cores e tipografia: ajuste em `css/style.css`.
- Slideshow, header fixo e animacoes: ajuste em `javascript/script.js`.

## Estrutura do projeto
- `index.html`: conteudo principal.
- `css/style.css`: visual e responsividade.
- `javascript/script.js`: interacoes (slideshow, efeitos, header fixo).
- `imagens/`: assets de imagem.
- `reserva/`: arquivos auxiliares.

## Tecnologias usadas
- HTML, CSS e JavaScript puro.
- Google Fonts e Font Awesome (icones).
- Google Ads tag para conversoes.
